                             AllBinary Platform Game Template Parent

What is it?

This Repository contains the main AllBinary Platform Game Template Parent for the AllBinary Platform.

You will need to build this before building the "AllBinary Platform Game Template" which is the Example 
for for the AllBinary Platform.

The AllBinary Platform Game Template repo that uses this is at: 

https://github.com/AllBinary/AllBinary-Platform-Game-Template
